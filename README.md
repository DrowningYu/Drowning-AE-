# DrowningYu.jsx
自己写的一些常用小脚本

目前可操作功能：

1. 生成一个模拟的An形式的帧选择器 该功能可以对选定图层的选定属性的关键帧进行绑定 并重复调用   效果展示视频：https://www.bilibili.com/video/BV1LV25YWEJn/    
2. 渲染当前帧至桌面
3. 自动的图层随动效果
4. 使用PS给当前合成生成法线图序列
5. 网易云歌词生成器 自动获取网易云歌词 生成合成自动对轴  视频演示效果：https://www.bilibili.com/video/BV1uJtPe9E1Q/
6.生成控制的空对象（摄像机暂时不支持）
7.拆分形状图层（没写 占位）
8.锚点重置（形状图层有bug）
9.表达式管理（没写 占位）
10.一键更新（正在测试）
